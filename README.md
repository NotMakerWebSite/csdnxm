### 三、功能描述
系统分为（身份）：用户、管理员。

**用户功能：**
1. 登录
2. 注册
3. 选择展览室
4. 预约场次
5. 预约pdf导出
6. 安全退出


**管理员功能：**
1. 登录
2. 用户信息管理
3. 修改预约信息
4. 修改用户信息
5. 退出登录
